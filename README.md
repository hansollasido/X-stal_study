# X_stal_study
- 아주대 학습 공동체 스터디입니다.
- 혼자공부하는 머신러닝, 딥러닝 책을 팀원들과 같이 학습하였습니다.
- Tensorflow의 keras와 scikit-learn을 사용하여 머신러닝, 딥러닝 학습을 하였습니다.
